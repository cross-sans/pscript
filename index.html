Pscript Documentation

Introduction

Pscript is a simple programming language designed for beginners to learn programming concepts. It is inspired by Python and uses a syntax similar to Python's.

Getting Started

To use Pscript, you will need a text editor to write your code and a Pscript compiler to convert your code to Python. You can download the Pscript compiler from the same profile you reached this pagge.

Syntax

Pscript uses a syntax similar to Python's, with a few differences:

  • Statements are separated by newlines.
  • Indentation is used to define blocks of code.
  • Variables are created using the "set" keyword.
  • Arrays are created using the "arr" keyword.
  • Functions are created using the "func" keyword.
  • Messages can be printed to the console using the ">:" and ":<" delimiters.
  • Comments can be added using the "<comment>" syntax.

Examples

Here are some examples of Pscript code:

class main[]

    set age to int(input("What is your age? "))
    print(>:"Hello, " + name + "! You are " + str(age) + " years old.":<)
end[]

Conclusion

Pscript is a simple programming language that can help beginners learn programming concepts. It is easy to learn and use, and can be a great stepping stone to more advanced languages.
