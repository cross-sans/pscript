Pscript Basics

Every instance of Pscript, whether you choose Java, JavaScript or legacy (Python/original), needs a main[] and end[] on it. This is because the compiler of choice uses it as the starting point for your code.

Another thing to note is that the Python version of Pscript has its own editor that is designed to work directly with the compiler. You can find it in the exe folder on GitHub, but you will need to extract the lib.zip file.

Pscript-Java

Pscript-Java is a version of Pscript that compiles Pscript code into Java bytecode. This allows Pscript code to run on any device that has the Java Virtual Machine (JVM) installed.

Pscript-JS

Pscript-JS is a version of Pscript that compiles Pscript code into JavaScript code. This allows Pscript code to run on any modern web browser.

Pscript legacy

Pscript legacy is the original version of Pscript that is compiled directly into Python.
